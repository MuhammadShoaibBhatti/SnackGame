# Snake Game (Java Swing)

A simple Snake Game developed in Java using Swing and AWT.

## Features

- Classic Snake gameplay
- Score tracking
- Random apple generation
- Collision detection
- Keyboard controls
- Game Over screen

## Technologies Used

- Java
- Swing
- AWT

## Controls

| Key | Action |
|------|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

## How to Run

### Compile
javac SnakeGame.java
java SnakeGame

Eat apples to increase your score.
Each apple increases the snake length.
